# Landing Page — Marcus Souza Advogado

Landing page estática feita em HTML + CSS + JavaScript.

## Arquivos principais
- `index.html`: estrutura da página.
- `style.css`: layout responsivo, cores, cards, gráficos e Visual Law.
- `script.js`: menu mobile, abas de áreas, validação do formulário e botão do WhatsApp.
- `obrigado.html`: página de confirmação para formulário Netlify.
- `assets/`: logomarca e imagens otimizadas.

## Captação de leads
O formulário foi configurado com `data-netlify="true"`. Ao publicar no Netlify, os cadastros podem aparecer no painel **Forms** do projeto.

Também existe o botão **Enviar dados pelo WhatsApp**, que gera uma mensagem automática com os dados preenchidos.

## Atenção antes de publicar
Recomenda-se inserir o número da OAB/UF no rodapé ou onde preferir antes da publicação definitiva.
